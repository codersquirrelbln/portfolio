Visa Calculator

The visa calculator enables travellers to calculate one or multiple trips to a country or visa zone such as the Schengen Zone. The user can insert individual visa regulations such as ho many days out if a time frame you may be in the country. It informs the user how many days they can still use up or have to remove to not overstay the visit. This project was built using JavaScript and made use of the flatpickr as a datetime picker and is still a work in progress…

ElderMe

ElderMe is a platform for senior users to find volunteers for activities such as going to the theatre, meeting for a cup of tea or taking a walk in the park. The project was built within two weeks using Ruby on Rail in collaboration with front end developers Ronan Recune and Monther Al Gbawi.

Visa Calculator

The visa calculator enables travellers to calculate one or multiple trips to a country or visa zone such as the Schengen Zone. The user can insert individual visa regulations such as ho many days out if a time frame you may be in the country. It informs the user how many days they can still use up or have to remove to not overstay the visit. This project was built using JavaScript and made use of the flatpickr as a datetime picker in collaboration with front end developer Ronan Recune.

Flashcard App

This flashcards app helps learning new words in a foreign language, manually add new words as well as a picture to go along with it. The picture can either be taken individually or inserted from sources off the internet. This project will be build in Ruby on Rails in collaboration with designer Stephanie Hawkins.
